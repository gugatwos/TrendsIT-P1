# Meu Portfólio Profissional Interativo

Este projeto foi desenvolvido como parte da Fase 1 - Projeto 1.

## Objetivo

Construir um site estático, responsivo e profissional para apresentar minha trajetória, minhas habilidades iniciais em tecnologia e meus projetos em desenvolvimento.

## Tecnologias utilizadas

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Git
- GitHub Pages

## Funcionalidades

- Seção de apresentação pessoal
- Seção sobre mim
- Lista de habilidades
- Galeria com projetos
- Links para contato
- Layout responsivo para celular e desktop
